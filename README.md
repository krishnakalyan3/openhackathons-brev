# Brev Containers for Open Hackathons

A curated collection of Docker Compose configurations designed for OpenHackathon projects on the `Brev` platform. This repository provides pre-configured AI and ML environments that accelerate development workflows, making it effortless to launch and experiment with powerful frameworks.
